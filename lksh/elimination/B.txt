По принципу Дирихле, если одна из последовательностей имеет длину меньше, чем n / 2 (округленное вниз),
то вторая последовательность будет иметь длину больше либо равную n / 2 (округленное вверх).

Рассмотрим равенство x + y = n + 1,
тогда x * y будет достигать максимального значения при abs(x - y) минимальном.
То есть, x * y будет достигать минимального значения, когда числа будут максимально удаленны друг от друга.
Рассмотрим этот пример: x = 1, y = n. x * y = n, то есть неравенство x * y >= n выполняется.
Если мы начнем менять элементы местами, неравенство будет сохранять истинность, т.к. даже если при определенном действии уменьшится одна из величин,
а вторая не увеличится, опять же, по принципу Дирихле.
